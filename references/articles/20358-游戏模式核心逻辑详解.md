# 游戏模式核心逻辑详解
##  系统职责

UGCGameMode 是整个团队竞技模式的核心控制器，负责：

* ✅ 游戏流程状态管理（准备→战斗→结算）
* ✅ 玩家登录/退出事件处理
* ✅ 队伍分配与切换逻辑
* ✅ 击杀事件与分数统计
* ✅ 复活机制管理
* ✅ 胜负条件判定
* ✅ 初始装备发放

##  核心数据结构

### 配置参数（CFG\_ 开头）
|参数|类型|说明|默认值|取值范围|
|-----|-----|-----|-----|-----|
|CFG\_IsEnablePlayerWeaponConfig|bool|是否允许玩家自定义装备|true| * |
|CFG\_DefaultEquipList|struct|默认装备列表| * | * |
|CFG\_IsEnablePlayerTeamSelect|bool|是否允许玩家选择队伍|true| * |
|CFG\_PrepareTime|float|准备时间（秒）|15|10-60|
|CFG\_BattleTime|float|战斗时间（秒）|300|90-600|
|CFG\_ResultTime|float|结算时间（秒）|30| * |
|CFG\_MaxTeamMember|int32|每队最大人数|4|1-4|
|CFG\_WinScore|int32|胜利分数|50|10-200|
|CFG\_RespawnInterval|float|复活等待时间（秒）|3|0.5-10|
|CFG\_KillScore|int32|击杀得分|1|≥0|
|CFG\_EquipSelectRange|struct|装备选择范围| * | * |

##  生命周期函数

### ReceiveBeginPlay()

**调用时机**：游戏模式初始化时

**主要功能**：

1. 初始化所有配置参数的 Clamp 值
2. 订阅玩家死亡事件 PawnDefeat
3. 初始化全局数据结构

```
function UGCGameMode:ReceiveBeginPlay()
    ugcprint("UGCGameMode:ReceiveBeginPlay")
    
    -- 初始化所有 Clamp 值
    self:InitAllClampNumAtGeneralRangeSettings()
    
    -- 订阅死亡事件
    UGCGenericMessageSystem.ListenGlobalMessage(
        self, 
        UGCGenericMessageSystem.Messages.UGC.PlayerPawn.PawnDefeat, 
        self, 
        self.OnPawnDefeat
    )
end
```

### InitAllClampNumAtGeneralRangeSettings()

**功能**：将配置参数限制在合理范围内，确保游戏平衡

**处理流程**：

1. **准备时间**：限制在 \[10, 60] 秒之间
2. **复活等待时间**：限制在 \[0.5, 10] 秒之间
3. **战斗时间**：限制在 \[90, 600] 秒之间
4. **每队人数**：限制在 \[1, 8] 人之间
5. **胜利分数**：限制在 \[10, 200] 分之间
6. **击杀得分**：≥ 0
7. **装备选择范围**：同步到 GameState
8. **队伍选择开关**：同步到 GameState
9. **装备配置开关**：同步到 GameState

```
function UGCGameMode:InitAllClampNumAtGeneralRangeSettings()
    -- 准备时间 Clamp
    self.GeneralGameSettings.ClampPrepareTime = math.max(
        self.GeneralRangeSettings.MinReadyTime, 
        math.min(self.GeneralRangeSettings.MaxReadyTime, self.CFG_PrepareTime)
    )
    
    -- 每队人数 Clamp
    self.GeneralGameSettings.ClampPlayerNum = math.max(
        self.GeneralRangeSettings.MinPlayerNumPerTeam, 
        math.min(self.GeneralRangeSettings.MaxPlayerNumPerTeam, self.CFG_MaxTeamMember)
    )
    self.GameState.TeamMaxPlayerNum = self.GeneralGameSettings.ClampPlayerNum
    UnrealNetwork.RepLazyProperty(self.GameState, "TeamMaxPlayerNum")
    
    -- 胜利分数 Clamp
    self.GeneralGameSettings.ClampWinScore = math.max(
        self.GeneralRangeSettings.MinTargetScore, 
        math.min(self.GeneralRangeSettings.MaxTargetScore, self.CFG_WinScore)
    )
    self.GameState.WinScore = self.GeneralGameSettings.ClampWinScore
    UnrealNetwork.RepLazyProperty(self.GameState, "WinScore")
    
    -- ... 其他参数初始化
end
```

##  玩家事件处理

### UGC\_PlayerLoginEvent(PlayerController)

**调用时机**：玩家加入房间时

**功能**：

1. 将玩家加入玩家列表
2. 设置初始队伍为 ETeamType.Initial
3. 第一个玩家加入后启动准备阶段

```
function UGCGameMode:UGC_PlayerLoginEvent(PlayerController)
    ugcprint("UGCGameMode:UGC_PlayerLoginEvent PlayerKey="..PlayerController.PlayerKey)
    
    -- 检查玩家是否已存在
    for _, key in ipairs(self.PlayerKeyList) do
        if key == PlayerController.PlayerKey then
            return
        end
    end
    
    -- 设置初始队伍
    UGCTeamSystem.ChangePlayerTeamID(PlayerController.PlayerKey, ETeamType.Initial)
    
    -- 加入玩家列表
    table.insert(self.PlayerKeyList, PlayerController.PlayerKey)
    
    -- 第一个玩家加入后开始游戏
    if self.GameState.CurrentGameStatus == EGameStatus.Initial then
        self.GameState:SetStatus(EGameStatus.Prepare, self.GeneralGameSettings.ClampPrepareTime)
    end
end
```

##  状态流转

### GotoNextStatus()

**功能**：处理游戏阶段切换逻辑

**流程图**

![企业微信截图_1775702937460.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/TSbYx%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_1775702937460.png)

**代码实现**：

```
function UGCGameMode:GotoNextStatus()
    if self.GameState.CurrentGameStatus == EGameStatus.Prepare then
        -- 为未选队伍的玩家随机分配队伍
        self:RandomTeamForPlayerWithoutTeam()
        
        -- 检查游戏是否结束
        if self:CheckGameEnd() then
            self.GameState:SetStatus(EGameStatus.Result, self.GeneralGameSettings.ClampFightTime)
        else
            -- 进入战斗阶段
            self.GameState:SetStatus(EGameStatus.Battle, self.GeneralGameSettings.ClampFightTime)
            
            -- 重建所有玩家
            for i,v in ipairs(self.GameState.TeamCurPlayerKeys) do
                for k,PlayerKey in pairs(v) do
                    local PC = UGCGameSystem.GetPlayerControllerByPlayerKey(PlayerKey)
                    if PC and PC.Pawn then
                        PC.Pawn:SetLifeSpan(0.001)
                    end
                    UGCGameSystem.RespawnPlayer(PlayerKey)
                end
            end
            
            -- 初始化全部玩家分数
            self.GameState:InitAllPlayerScore()
        end
        
    elseif self.GameState.CurrentGameStatus == EGameStatus.Battle then
        -- 进入结算阶段
        self.GameState:SetStatus(EGameStatus.Result, self.CFG_ResultTime)
    end
end
```

##  队伍管理

### PlayerChangeTeam(PlayerKey, NewTeamID)

**功能**：切换玩家队伍

**参数**：

* PlayerKey：玩家标识
* NewTeamID：目标队伍 ID（Red=1, Blue=2）

**逻辑**：

1. 验证队伍 ID 有效性
2. 检查目标队伍是否已满员
3. 避免重复切换相同队伍
4. 调用系统 API 切换队伍
5. 更新队伍数据
6. 复活到新的出生点

```
function UGCGameMode:PlayerChangeTeam(PlayerKey, NewTeamID)
    ugcprint("UGCGameMode:PlayerChangeTeam "..tostring(PlayerKey).." NewTeamID = "..tostring(NewTeamID))
    
    -- 验证队伍 ID
    if NewTeamID ~= ETeamType.Red and NewTeamID ~= ETeamType.Blue then
        ugcprint("UGCGameMode:PlayerChangeTeam Team with Invalid TeamID")
        return
    end
    
    -- 检查队伍是否已满员
    local TeamMemberNum = UGCTeamSystem.GetTeamSizeByID(NewTeamID)
    if TeamMemberNum >= self.GeneralGameSettings.ClampPlayerNum then
        ugcprint("UGCGameMode:PlayerChangeTeam Team without pos")
        return
    end
    
    local PC = UGCGameSystem.GetPlayerControllerByPlayerKey(PlayerKey)
    
    -- 避免重复切换
    if PC and PC.TeamID == NewTeamID then
        ugcprint("UGCGameMode:PlayerChangeTeam Team with Same TeamID")
        self.GameState:UpdateTeamData()
        return
    end
    
    -- 切换队伍
    UGCTeamSystem.ChangePlayerTeamID(PlayerKey, NewTeamID)
    self.GameState:UpdateTeamData()
    
    -- 复活到新出生点
    if PC and PC.Pawn then
        PC.Pawn:SetLifeSpan(0.001)
    end
    UGCGameSystem.RespawnPlayer(PlayerKey)
end
```

### RandomTeamForPlayerWithoutTeam()

**功能**：为未选择队伍的玩家随机分配队伍

**分配策略**：优先分配到人数较少的队伍，确保平衡

```
function UGCGameMode:RandomTeamForPlayerWithoutTeam()
    for i=1, #self.PlayerKeyList do
        local PlayerKey = self.PlayerKeyList[i]
        local PC = UGCGameSystem.GetPlayerControllerByPlayerKey(PlayerKey)
        
        if PC and PC.TeamID == ETeamType.Initial then
            -- 优先分配玩家少的队伍
            if #self.GameState.TeamCurPlayerKeys[ETeamType.Red] < 
               #self.GameState.TeamCurPlayerKeys[ETeamType.Blue] then
                self:PlayerChangeTeam(PlayerKey, ETeamType.Red)
            else
                self:PlayerChangeTeam(PlayerKey, ETeamType.Blue)
            end
        end
    end
end
```

##  战斗逻辑

### OnPawnDefeat(VictimPlayerKey, InstigatorPlayerKey, DamageType)

**调用时机**：玩家死亡时（通过事件订阅）

**功能**：

1. 检查是否在结算阶段（是则不处理）
2. 更新击杀者和被击杀者的战绩
3. 安排复活
4. 检查胜利条件

```
function UGCGameMode:OnPawnDefeat(VictimPlayerKey, InstigatorPlayerKey, DamageType)
    -- 结算阶段不计分
    if EGameStatus.Result == self.GameState.CurrentGameStatus then
        return
    end
    
    local Killer = UGCGameSystem.GetPlayerControllerByPlayerKey(InstigatorPlayerKey)
    local VictimPlayer = UGCGameSystem.GetPlayerControllerByPlayerKey(VictimPlayerKey)
    
    -- 更新积分
    self.GameState:UpdateScore(Killer, VictimPlayer)
    
    -- 复活
    if VictimPlayer then
        UGCTimerUtility.CreateLuaTimer(
            self.GeneralGameSettings.ClampRespawnWaitingTime, 
            function()
                UGCGameSystem.RespawnPlayer(VictimPlayer.PlayerKey)
            end,
            false
        )
    end
    
    -- 检测结算条件
    if self:CheckGameEnd() then
        self.GameState:SetStatus(EGameStatus.Result, self.CFG_ResultTime)
    end
end
```

### CheckGameEnd()

**功能**：检查游戏是否满足结束条件

**胜利条件**：

1. 任意队伍人数为 0
2. 任意队伍达到胜利分数

```
function UGCGameMode:CheckGameEnd()
    -- 检查队伍是否有人
    if #self.GameState.TeamCurPlayerKeys[ETeamType.Red] == 0 or 
       #self.GameState.TeamCurPlayerKeys[ETeamType.Blue] == 0 then
        return true
    end
    
    -- 检查是否达到胜利分数
    if self.GameState.TeamScore[ETeamType.Red] >= self.GameState.WinScore or 
       self.GameState.TeamScore[ETeamType.Blue] >= self.GameState.WinScore then
        return true
    end
    
    return false
end
```

### AddInitalItem(Pawn, ItemID, TotalCount)

**功能**：添加单个物品到背包，支持堆叠

**处理逻辑**：

1. 检查物品 ID 有效性
2. 如果数量 ≤ 最大堆叠数，直接添加
3. 如果数量 > 最大堆叠数，拆分为多个堆叠

```
function UGCGameMode:AddInitalItem(Pawn, ItemID, TotalCount)
    local ItemConfig = UGCItemSystem.GetItemData(ItemID)
    
    -- 检查物品有效性
    if ItemConfig.ItemID == 0 then
        print(string.format("UGCGameMode:AddInitalItem ItemID=%d not found", ItemID))
        return
    end
    
    -- 数量不超限直接添加
    if TotalCount <= ItemConfig.MaxCount then
        UGCBackPackSystem.AddItem(Pawn, ItemID, TotalCount)
        return
    end
    
    -- 数量超限拆分堆叠
    local StackCount = math.floor(TotalCount / ItemConfig.MaxCount)
    local LastStackCount = TotalCount % ItemConfig.MaxCount
```

**查找逻辑**：

```
        TeamID = 1
    end
    
    local PlayerStartClass = UGCObjectUtility.LoadClass(
        "/Game/BluePrints/Player/PlayerStart/BP_STPlayerStart.BP_STPlayerStart_C"

    for i=1, #PlayerStartList do
```
