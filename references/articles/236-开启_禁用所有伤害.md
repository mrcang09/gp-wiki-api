# 开启/禁用所有伤害

- 在GameModez中设置EnableDamage

![001.00a5c143[1].png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/001.00a5c143%5B1%5D.png)

- 代码中修改示例

```lua
local Action_SetEnableDamage = 
{
	IsEnableDamage = false;
}


function Action_SetEnableDamage:Execute()
	ugcprint(string.format("Action_SetEnableDamage:Execute IsEnableDamage[%s]", tostring(self.IsEnableDamage)));

	UGCGameSystem.GameMode.bEnableDamage = self.IsEnableDamage;

	return true;
end


return Action_SetEnableDamage
```
