# 事件系统与枚举定义详解
##  系统概述

事件系统和枚举定义是团队竞技工程的核心基础设施，提供：

* ✅ 游戏阶段枚举（准备/战斗/结算）
* ✅ 队伍类型枚举（初始/红队/蓝队）
* ✅ 客户端事件定义（游戏状态变更、队伍数据更新、战绩更新等）
* ✅ 事件订阅/发布机制
* ✅ 服务器→客户端事件通信

***

##  枚举定义

### CTCompetitionEnum.lua

**文件路径**：Script/Common/CTCompetitionEnum.lua

**作用**：定义游戏核心枚举类型

### 游戏阶段枚举（EGameStatus）

```
EGameStatus = 
{
    Initial = 0,  -- 未赋值（初始状态）
    Prepare = 1,  -- 准备阶段
    Battle = 2,   -- 战斗阶段
    Result = 3    -- 结算阶段
}
```

**原代码路径**：Script/Common/CTCompetitionEnum.lua:3-9

**使用方式演示**：

```
-- 检查当前阶段
if self.GameState.CurrentGameStatus == EGameStatus.Prepare then
    -- 允许玩家选择队伍和装备
end

-- 切换到战斗阶段
self.GameState:SetStatus(EGameStatus.Battle, 300)

-- 判断是否为结算阶段
if self.GameState.CurrentGameStatus == EGameStatus.Result then
    -- 显示结算界面
end
```

##  客户端事件定义

### CTCompetitioClientEvent.lua

**文件路径**：Script/Common/CTCompetitioClientEvent.lua

**作用**：定义客户端可接收的游戏事件

### 事件列表

|事件ID|事件名称|说明|
|-----|-----|-----|
|101|GameStatusChanged|游戏状态变更|
|102|NextStatusTimeStampChanged|阶段结束时间变更|
|103|TeamDataChanged|队伍数据变更|
|104|TeamUIDChanged|队伍 UID 变更|
|105|PlayerScoreChanged|玩家战绩变更|
|106|TeamScoreChanged|队伍积分变更|
|107|GameEndEvent|游戏结束事件|
|108|ShowCommonTipsUI|显示提示消息|
|109|EquipSelectRangeChanged|装备选择范围变更|

**原代码路径**：Script/Common/CTCompetitioClientEvent.lua:4-53

### 事件详解

#### GameStatusChanged

**说明**：游戏状态变更事件

**参数**：

* CurrentGameStatus：新的游戏状态（EGameStatus 枚举）

**触发时机**：

* 进入准备阶段
* 进入战斗阶段
* 进入结算阶段

**使用方式演示**：

```
-- 在 UI 脚本中订阅事件
function MyWidget:OnWidgetShow()
    UGCEventSystem:RegisterEvent(
        CTCompetitioClientEvent.GameStatusChanged, 
        self, 
        self.OnGameStatusChanged
    )
end

function MyWidget:OnGameStatusChanged(NewStatus)
    if NewStatus == EGameStatus.Prepare then
        -- 显示准备阶段 UI
        self:ShowPrepareUI()
    elseif NewStatus == EGameStatus.Battle then
        -- 显示战斗阶段 UI
        self:ShowBattleUI()
    elseif NewStatus == EGameStatus.Result then
        -- 显示结算界面
        self:ShowResultUI()
    end
end
```

***

#### NextStatusTimeStampChanged

**说明**：阶段结束时间变更事件

**参数**：

* NextStatusTimeStamp：下一阶段的结束时间（世界时间戳）

**使用方式演示**：

```
function MyWidget:OnNextStatusTimeStampChanged(TimeStamp)
    local CurrentTime = self.GameState:GetWorldTime()
    local RemainingTime = TimeStamp - CurrentTime
    
    -- 更新倒计时显示
    self.CountdownText:SetText(string.format("%02d:%02d", 
        math.floor(RemainingTime / 60), 
        math.floor(RemainingTime % 60)))
end
```

#### TeamDataChanged

**说明**：队伍数据变更事件（玩家 Key 列表）

**参数**：

```
TeamCurPlayerKeys = {
    [ETeamType.Red] = {PlayerKey1, PlayerKey2, ...},
    [ETeamType.Blue] = {PlayerKey3, PlayerKey4, ...}
}
```

**使用方式演示**：

```
function MyWidget:OnTeamDataChanged(TeamData)
    -- 更新红队玩家列表
    for _, PlayerKey in ipairs(TeamData[ETeamType.Red]) do
        local PS = UGCGameSystem.GetPlayerStateByPlayerKey(PlayerKey)
        if PS then
            self:AddPlayerToRedList(PS.PlayerName)
        end
    end
    
    -- 更新蓝队玩家列表
    for _, PlayerKey in ipairs(TeamData[ETeamType.Blue]) do
        local PS = UGCGameSystem.GetPlayerStateByPlayerKey(PlayerKey)
        if PS then
            self:AddPlayerToBlueList(PS.PlayerName)
        end
    end
end
```

#### PlayerScoreChanged

**说明**：玩家战绩变更事件

**参数**：

```
PlayerScore = {
    [PlayerKey1] = {
        KillCount = 5,
        UID = "10001",
        DeathCount = 2,
        TeamID = 1
    },
    [PlayerKey2] = {
        KillCount = 3,
        UID = "10002",
        DeathCount = 4,
        TeamID = 2
    }
}
```

**使用方式演示**：

```
function MyWidget:OnPlayerScoreChanged(PlayerScore)
    -- 清空现有数据
    self:ClearScoreBoard()
    
    -- 遍历所有玩家战绩
    for PlayerKey, ScoreData in pairs(PlayerScore) do
        local PS = UGCGameSystem.GetPlayerStateByPlayerKey(PlayerKey)
        if PS then
            self:AddScoreEntry({
                Name = PS.PlayerName,
                KillCount = ScoreData.KillCount,
                DeathCount = ScoreData.DeathCount,
                TeamID = ScoreData.TeamID
            })
        end
    end
end
```

**说明**：队伍积分变更事件

```
TeamScore = {
```

**使用方式演示**：

```
function MyWidget:OnTeamScoreChanged(TeamScore)
    -- 更新红队积分显示
    self.RedTeamScoreText:SetText(tostring(TeamScore[ETeamType.Red]))
    
    -- 更新蓝队积分显示
    self.BlueTeamScoreText:SetText(tostring(TeamScore[ETeamType.Blue]))
    
    -- 检查是否达到胜利分数
    if TeamScore[ETeamType.Red] >= self.GameState.WinScore then
        self:ShowTeamScoreWarning(ETeamType.Red)
    elseif TeamScore[ETeamType.Blue] >= self.GameState.WinScore then
        self:ShowTeamScoreWarning(ETeamType.Blue)
    end
end
```

#### GameEndEvent

**说明**：游戏结束事件

**参数**：

```
BattleResult = {
    WinTeam = 1,  -- 0=平局, 1=红队, 2=蓝队
    TeamScore = {
        [1] = 50,
        [2] = 30
    },
    PlayerScore = {
        [PlayerKey1] = {
            KillCount = 5,
            DeathCount = 2,
            AssistCount = 3,
            TeamID = 1,
            UID = "10001"
        },
        ...
    }
}
```

**使用方式演示**：

```
function MyWidget:OnGameEndEvent(BattleResult)
    -- 判定获胜队伍
    if BattleResult.WinTeam == ETeamType.Red then
        self.WinnerText:SetText("红队获胜！")
        self.WinnerIcon:SetColor(FLinearColor(1.0, 0.2, 0.2, 1.0))  -- 红色
    elseif BattleResult.WinTeam == ETeamType.Blue then
        self.WinnerText:SetText("蓝队获胜！")
        self.WinnerIcon:SetColor(FLinearColor(0.2, 0.5, 1.0, 1.0))  -- 蓝色
    else
        self.WinnerText:SetText("平局！")
        self.WinnerIcon:SetColor(FLinearColor(0.8, 0.8, 0.8, 1.0))  -- 灰色
    end
    
    -- 显示最终战绩
    self:ShowFinalScoreboard(BattleResult.PlayerScore)
    
    -- 判定 MVP（击杀数最多的玩家）
    local MVPPlayerKey = nil
    local MaxKillCount = 0
    
    for PlayerKey, ScoreData in pairs(BattleResult.PlayerScore) do
        if ScoreData.KillCount > MaxKillCount then
            MaxKillCount = ScoreData.KillCount
            MVPPlayerKey = PlayerKey
        end
    end
    
    if MVPPlayerKey then
        local PS = UGCGameSystem.GetPlayerStateByPlayerKey(MVPPlayerKey)
        if PS then
            self:MVPPlayerText:SetText("MVP: " .. PS.PlayerName)
        end
    end
end
```

##  事件系统使用

### 事件订阅

**方式 1：在 Widget 中订阅**

**使用方式演示**：

```
function MyWidget:OnWidgetShow()
    -- 订阅游戏状态变更事件
    UGCEventSystem:RegisterEvent(
        CTCompetitioClientEvent.GameStatusChanged, 
        self, 
        self.OnGameStatusChanged
    )
    
    -- 订阅队伍数据变更事件
    UGCEventSystem:RegisterEvent(
        CTCompetitioClientEvent.TeamDataChanged, 
        self, 
        self.OnTeamDataChanged
    )
    
    -- 订阅玩家战绩变更事件
    UGCEventSystem:RegisterEvent(
        CTCompetitioClientEvent.PlayerScoreChanged, 
        self, 
        self.OnPlayerScoreChanged
    )
end
```

***

##  开发建议

### 1. 添加自定义事件

**步骤**：

1. 在 CTCompetitioClientEvent.lua 中定义新事件 ID
2. 在 UGCGameState.lua 中发布事件
3. 在 UI 脚本中订阅事件

**示例：添加击杀通知事件**：

```
-- 1. 在 CTCompetitioClientEvent.lua 中添加
CTCompetitioClientEvent.KillNotification = 110

-- 参数格式
--[[
params:
  ├─ KillerName: "张三"
  ├─ VictimName: "李四"
  ├─ WeaponName: "M416"
  └─ IsHeadshot: true
--]]

-- 2. 在 UGCGameState.lua 中发布事件
function UGCGameState:UpdateScore(Killer, VictimPlayer)
    -- 原有战绩更新逻辑...
    
    -- 发布击杀通知事件
    if Killer and VictimPlayer then
        local KillerPS = UGCGameSystem.GetPlayerStateByPlayerKey(Killer.PlayerKey)
        local VictimPS = UGCGameSystem.GetPlayerStateByPlayerKey(VictimPlayer.PlayerKey)
        
        if KillerPS and VictimPS then
            local KillData = {
                KillerName = KillerPS.PlayerName,
                VictimName = VictimPS.PlayerName,
                WeaponName = "M416",  -- 可根据实际情况获取
                IsHeadshot = false
            }
            
            UGCEventSystem:SendEvent(CTCompetitioClientEvent.KillNotification, KillData)
        end
    end
end

-- 3. 在 UI 脚本中订阅事件
function MyWidget:OnWidgetShow()
    UGCEventSystem:RegisterEvent(
        CTCompetitioClientEvent.KillNotification, 
        self, 
        self.OnKillNotification
    )
end

function MyWidget:OnKillNotification(KillData)
    local Message = string.format("%s 击杀了 %s", KillData.KillerName, KillData.VictimName)
    if KillData.IsHeadshot then
        Message = Message .. " [爆头！]"
    end
    
    self:AddKillMessage(Message)
end
```

```
            end
        )
    end
    
    -- ... 原有 UI 切换逻辑
end
```

### 3. 添加新的队伍类型

**步骤**：

1. 在 CTCompetitionEnum.lua 中添加新枚举值
2. 在 UGCGameState.lua 中初始化新队伍数据
3. 修改所有队伍相关的逻辑

**示例：添加第三队伍（绿队）**：

```
-- 1. 在 CTCompetitionEnum.lua 中添加
ETeamType =
{
    Initial = 0,
    Red = 1,
    Blue = 2,
    Green = 3  -- 绿队
}

-- 2. 在 UGCGameState.lua 中初始化
UGCGameState.TeamCurPlayerKeys[ETeamType.Red] = {}
UGCGameState.TeamCurPlayerKeys[ETeamType.Blue] = {}
UGCGameState.TeamCurPlayerKeys[ETeamType.Green] = {}  -- 新增
```
