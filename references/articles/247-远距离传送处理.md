# 远距离传送处理

- 如果使用了K2_SetLocation传送玩家的方式，需要修改成以下方式
- 防止由于一些原因导致的网络拉扯，传送失败
- 防止过远传送触发保护机制打印大量异常log造成卡顿

``` lua
PlayerPawn.bShouldDumpCallstackWhenMovingfast = false
PlayerPawn:SetClientLocationOrRotation(Loc, Rot, true, false)
PlayerPawn.bShouldDumpCallstackWhenMovingfast = true
```
