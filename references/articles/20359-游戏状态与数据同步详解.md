# 游戏状态与数据同步详解
##  系统职责

UGCGameState 是游戏状态的中央管理器，负责：

* ✅ 游戏阶段状态管理（准备/战斗/结算）
* ✅ 队伍数据同步（服务器→客户端）
* ✅ 战绩数据同步（服务器→客户端）
* ✅ UI 显示控制（根据阶段切换界面）
* ✅ 数据复制属性定义
* ✅ 时间管理

##  核心数据结构

##  游戏阶段管理

### SetStatus(InStatus, InDuration)

**功能**：切换游戏阶段并设置持续时间

**参数**：

* InStatus：目标阶段（EGameStatus 枚举）
* InDuration：持续时间（秒）

**处理逻辑**：

1. 更新当前阶段
2. 计算阶段结束时间
3. 根据阶段设置玩家无敌状态
4. 结算阶段准备并下发结算数据
5. 同步数据到客户端

```
function UGCGameState:SetStatus(InStatus, InDuration)
    self.CurrentGameStatus = InStatus
    self.NextStatusTimeStamp = self:GetWorldTime() + InDuration
    
    ugcprint("UGCGameMode:SetStatus InStatus="..tostring(InStatus).." InDuration="..tostring(InDuration).." NextStatusTimeStamp="..tostring(self.NextStatusTimeStamp))
    
    -- 根据阶段设置无敌状态
    if self.CurrentGameStatus == EGameStatus.Prepare then
        local AllPawn = UGCGameSystem.GetAllPlayerPawn()
        for _, Pawn in ipairs(AllPawn) do
            UGCPlayerPawnSystem.SetIsInvincible(Pawn, true)
        end
    else
        local AllPawn = UGCGameSystem.GetAllPlayerPawn()
        for _, Pawn in ipairs(AllPawn) do
            UGCPlayerPawnSystem.SetIsInvincible(Pawn, false)
        end
    end
    
    -- 结算阶段准备
    if self.CurrentGameStatus == EGameStatus.Result then
        -- 判定胜利队伍
        if self.TeamScore[ETeamType.Red] == self.TeamScore[ETeamType.Blue] then
            self.BattleResult.WinTeam = 0
        elseif self.TeamScore[ETeamType.Red] > self.TeamScore[ETeamType.Blue] then
            self.BattleResult.WinTeam = ETeamType.Red
        else
            self.BattleResult.WinTeam = ETeamType.Blue
        end
        
        -- 下发战绩数据
        self.BattleResult.PlayerScore = self.PlayerScore
        log_tree("UGCGameMode:Send BattleResult", self.BattleResult)
        UnrealNetwork.RepLazyProperty(self, "BattleResult")
    end
    
    -- 同步数据到客户端
    UnrealNetwork.RepLazyProperty(self, "CurrentGameStatus")
    UnrealNetwork.RepLazyProperty(self, "NextStatusTimeStamp")
end
```

```
function UGCGameState:UpdateTeamData()
    -- 更新红队数据
    self.TeamCurPlayerKeys[ETeamType.Red] = {}
    self.TeamCurPlayerUIDs[ETeamType.Red] = {}
    for k,v in ipairs(UGCTeamSystem.GetPlayerKeysByTeamID(ETeamType.Red) or {}) do

    -- 更新蓝队数据
```

##  战绩数据管理

### UpdateScore(Killer, VictimPlayer)

**功能**：更新击杀/死亡/助攻数据（服务器端）

**计分规则**：

* **击杀**：击杀者 KillCount + 1
* **死亡**：被击杀者 DeathCount + 1
* **助攻**：造成伤害的非击杀者 AssistCount + 1

```
function UGCGameState:UpdateScore(Killer, VictimPlayer)
    local PlayerScore = self.PlayerScore
    
    -- 击杀者加分（必须不同队）
    if Killer and VictimPlayer and Killer.TeamID ~= VictimPlayer.TeamID then
        self:TryInitPlayerScore(Killer)
        PlayerScore[Killer.PlayerKey].KillCount = PlayerScore[Killer.PlayerKey].KillCount + 1
    end
    
    -- 被击杀者记录死亡
    if VictimPlayer then
        self:TryInitPlayerScore(VictimPlayer)
        PlayerScore[VictimPlayer.PlayerKey].DeathCount = PlayerScore[VictimPlayer.PlayerKey].DeathCount + 1
        
        -- 助攻判定（造成伤害的其他玩家）
        if VictimPlayer.PlayerState and next(VictimPlayer.PlayerState.DamagePlayerKeys) ~= nil then
            for k,v in pairs(VictimPlayer.PlayerState.DamagePlayerKeys) do
                if k ~= Killer.PlayerKey and k ~= VictimPlayer.PlayerKey then
                    local AssistPC = UGCGameSystem.GetPlayerControllerByPlayerKey(k)
                    if AssistPC then
                        self:TryInitPlayerScore(AssistPC)
                        PlayerScore[k].AssistCount = PlayerScore[k].AssistCount + 1
                    end
                    break
                end
            end
        end
    end
    
    log_tree("UGCGameState:UpdateScore", PlayerScore)
    
    -- 同步数据
    UnrealNetwork.RepLazyProperty(self, "PlayerScore")
    
    -- 更新队伍积分
    self:UpdateTeamScore()
end
```

### UpdateTeamScore()

**功能**：计算队伍总积分

**计分公式**：

```
队伍积分 = Σ (玩家击杀数 × 击杀得分)
```

```
function UGCGameState:UpdateTeamScore()
    self.TeamScore[ETeamType.Red] = 0
    self.TeamScore[ETeamType.Blue] = 0
    
    for k,v in pairs(self.PlayerScore) do
        if v.TeamID == ETeamType.Blue or v.TeamID == ETeamType.Red then
            self.TeamScore[v.TeamID] = self.TeamScore[v.TeamID] + v.KillCount * self.KillScore
        end
    end
    
    log_tree("UGCGameState:UpdateTeamScore", self.TeamScore)
end
```

### TryInitPlayerScore(PC)

```
PlayerScore[PlayerKey] = {
```

```
function UGCGameState:TryInitPlayerScore(PC)
    local PlayerScore = self.PlayerScore
    
    if PlayerScore[PC.PlayerKey] == nil then
        PlayerScore[PC.PlayerKey] = {}
        PlayerScore[PC.PlayerKey].KillCount = 0
```

##  结算数据

### OnRep\_BattleResult()

**调用时机**：服务器同步 BattleResult 到客户端时

**功能**：

1. 根据玩家战绩计算队伍积分
2. 发送游戏结束事件

**数据结构**：

```
BattleResult = {
    WinTeam = ETeamType.Red,  -- 获胜队伍（0=平局, 1=红队, 2=蓝队）
    TeamScore = {
        [ETeamType.Red] = 50,
        [ETeamType.Blue] = 30
    },
    PlayerScore = {
        [PlayerKey1] = {
            KillCount = 5,
            DeathCount = 2,
            AssistCount = 3,
            TeamID = 1,
            UID = "10001"
        },
        ...
    }
}
```

```
function UGCGameState:OnRep_BattleResult()
    if next(self.BattleResult) == nil then
        return
    end
    
    -- 重新计算队伍积分（确保数据一致性）
    local TeamScore = {}
    TeamScore[ETeamType.Red] = 0
    TeamScore[ETeamType.Blue] = 0
    
    for k,v in pairs(self.BattleResult.PlayerScore) do
        if v.TeamID == ETeamType.Blue or v.TeamID == ETeamType.Red then
            TeamScore[v.TeamID] = TeamScore[v.TeamID] + v.KillCount * self.KillScore
        end
    end
    
    self.BattleResult.TeamScore = TeamScore
    log_tree("UGCGameMode:OnRep_BattleResult", self.BattleResult)
    
    -- 发送游戏结束事件
    UGCEventSystem:SendEvent(CTCompetitioClientEvent.GameEndEvent, self.BattleResult)
end
```

##  开发建议

### 1. 添加新的同步属性

如需添加新的同步属性：

1. 在 UGCGameState 中添加字段
2. 在 GetReplicatedProperties() 中注册
3. 创建 OnRep\_PropertyName() 回调函数
4. 在服务器端调用 UnrealNetwork.RepLazyProperty()

**使用方式演示**：

```
-- 1. 添加字段
UGCGameState.RoundScore = 0

-- 2. 注册同步属性
function UGCGameState:GetReplicatedProperties()
    return {
        ...
        {"RoundScore", "Lazy"}
    }
end

-- 3. 创建回调
function UGCGameState:OnRep_RoundScore()
    UGCEventSystem:SendEvent(CustomEvent.RoundScoreChanged, self.RoundScore)
end

-- 4. 服务器端调用
self.RoundScore = 100
UnrealNetwork.RepLazyProperty(self, "RoundScore")
```

### 2. 修改战绩计算规则

当前战绩规则在 UpdateScore() 中定义，可修改：

* 添加爆头加分
* 添加连杀奖励
* 修改助攻判定条件

**使用方式演示-添加爆头加分**

```
function UGCGameState:UpdateScore(Killer, VictimPlayer, DamageType)
    -- 原有逻辑...
    
    -- 添加爆头判断
    if DamageType == EHeadShot then
        self:TryInitPlayerScore(Killer)
        PlayerScore[Killer.PlayerKey].HeadshotCount = PlayerScore[Killer.PlayerKey].HeadshotCount or 0
        PlayerScore[Killer.PlayerKey].HeadshotCount = PlayerScore[Killer.PlayerKey].HeadshotCount + 1
    end
    
    -- ...
end
```

### 3. 优化同步性能

**技巧 1：减少不必要的同步**

```
-- 仅在数据变化时同步
if self.TeamScore[ETeamType.Red] ~= oldRedScore then
    UnrealNetwork.RepLazyProperty(self, "TeamScore")
end
```

**技巧 2：批量同步**

```
-- 一次性同步多个属性
UnrealNetwork.RepLazyProperty(self, {"PlayerScore", "TeamScore", "WinScore"})
```

**技巧 3：使用条件同步**

```
-- 仅同步给特定队伍
UnrealNetwork.RepLazyPropertyToTeam(self, "PlayerScore", ETeamType.Red)
```
