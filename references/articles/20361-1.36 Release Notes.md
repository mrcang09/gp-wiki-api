# 1.36 Release Notes

## 1.36.11.29254 更新说明

### 编辑器

**bug Fixes**

- 优化物品ID字段，可创建的物品种类上限提升至1000种
- 修复俯视角玩法模板中，玩家重生后BuffUI未及时移除的问题
- 修复俯视角玩法模板中，玩家重生后引力陷阱技能冷却时间未正确刷新的问题
- 修复俯视角玩法模板中，玩家的武器弹道会被队友阻挡的问题
- 修复俯视角玩法模板中，偶现飞行怪物因碰撞体积异常而被卡至空中的问题
- 修复玩法上传后，有概率出现进入对局时闪退的问题
- 修复怪物机甲兽三角龙仅能循环使用单一技能的问题
- 修复怪物机甲兽三角龙使用冲刺后被异常拉回起点的问题
- 修复创建空白枪械时异常附带默认配件的问题
- 修复枪械装配倍镜后，丢弃再拾取导致倍镜效果消失的问题
- 修复调试部分工程时服务端日志不完整或为空的问题
- 修复在移动端实机中怪物转向表现异常的问题
- 修复技能编辑器中动画吸附配置项未汉化的问题

## 1.36.11.29238 版本说明

### 编辑器

**新功能**

- 新增交互物蓝图，方便开发者快速制作可交互场景物件
- 新增Tween动画系统，支持位置、旋转、颜色等数值的平滑过渡，适用于UI动效与场景变换
- 新增Actor实例路径使用说明，方便开发者进行高效引用与操作对象
- 新增AIGC地形生成插件，大幅降低地形制作门槛
- 新增日志面板，提升日志排查效率
- 新增玩法兑换码，支持配置和使用玩法兑换码功能
- 新增Getter、Setter自动新建变量功能

**功能更新**

【物资系统】

**物品编辑器**
- 新增物品状态相关的GameplayTag
- 支持从消耗物品接入技能
- 支持从装备物品Handle配置挂接Actor
- 优化装备物品的技能接入
- 自定义Avatar装备支持预览偏移
- 配置物品时支持添加数据资产类型的资源
- 新增物品模板资源：

| 类型 | 模板名称 | 
| :------: | :------: | 
| 机枪 | M338轻机枪 |
| 手枪 | F57手枪 |
| 重型武器 | 铁拳火箭筒 |
| 特殊武器 | 信号枪 |
| 子弹物品 | 信号弹 |
| 子弹物品 | .338口径子弹 |

**背包系统**
- 支持为背包物品自定义品质效果

【技能系统】

**技能编辑器**
- 支持监听物品的使用状态
- 支持从服务端获取技能Task的Socket信息
- 进度条UITask支持自定义
- 优化技能Task生命周期
- 优化挥拳攻击，现在的挥拳攻击基于技能编辑器2.0实现
- 支持打断攀爬Tag ``PawnState.Action.Vault`` 的行为，修复状态改变Task对该Tag无效的问题
- 新增技能Task音效配置参数
- 优化AttachActor相关参数
- 新增技能模板资源：

| 类型 | 模板名称 | 
| :------: | :------: | 
| 主动技能-选取型技能 | 小鸡炸弹 |
| 主动技能-直接释放型技能 | 七星剑阵 |
| 主动技能-蓄力型技能 | 蓄力冲拳 |
| 主动技能-选取型技能 | 海神三叉戟 |

**Buff编辑器**

- BuffAction支持附加ActorAction
- 支持按类型Tag移除Buff
- 新增Buff音效配置参数
- 新增buff模板资源：

| 类型 | 模板名称 | 
| :------: | :------: | 
| 特殊效果-变身 | 怪物变身-光子鸡 |

【通用抛体】

- 抛体Action支持移除BuffAction
- 支持箭头组件
- 支持使用曲线进行抛体轨迹基础配置
- 支持配置最大反弹次数

【法术场】

- 支持多碰撞盒拼接

【UI系统】

**UI编辑器**

- 玩家血条、怪物血条新增缩放比参数
- 新增UI元件资源：

| 类型 | 模板名称 | 
| :------: | :------: | 
| 系统 | 强引导组件 |
| 系统 | 头像框组件 |
| 系统 | 条形进度条组件 |
| 系统 | 环形进度条组件 |

【载具系统】
- 编辑器资源新增单种载具的刷新点蓝图

【其他】
- 优化道路生成器功能
- 优化 [俯视角模板](https://developer.gp.qq.com/wikieditor/#/catalog/20289)
- [GM面板](https://developer.gp.qq.com/wikieditor/#/catalog/20109) 新增更多预置功能
- [GM面板](https://developer.gp.qq.com/wikieditor/#/catalog/20109) 已从主界面分离，隐藏主界面将不会影响GM面板的可视性
- 优化编辑器界面布局
- 新增即时属性概念
- Lua绑定自动以函数模板创建
- 优化从编辑器打开Lua的方式，打开对应Lua文件前会先打开工程对应的工作空间
- [特效编辑器](https://developer.gp.qq.com/wikieditor/#/catalog/20223) 进行了优化，操作更加方便高效
