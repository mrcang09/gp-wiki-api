## Simple Parallel

| 属性 | 描述 |
| :--- | :---: |
| **Finish Mode** | 后台树的结束模式 | 
| **Node Name** | 节点在行为树图中显示的名称 |

| Finish Mode | 描述 |
| :--- | :---: |
| **AbortBackground** | 主任务完成后，后台树的执行将立即中止 | 
| **WaitForBackground** | 主任务完成后，允许后台树继续执行直至完成 |
